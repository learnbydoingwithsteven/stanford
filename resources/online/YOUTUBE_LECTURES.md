## 💡 Tips for Learning from Videos

### Effective Viewing
1. **Take Notes**: Pause and write down key concepts
2. **Watch at 1.25-1.5x**: Save time without losing comprehension
3. **Do Assignments**: Find problem sets online
4. **Rewatch**: Complex topics need multiple viewings
5. **Supplement**: Use textbooks alongside videos

### Best Practices
- **Schedule**: Set regular viewing times
- **Environment**: Minimize distractions
- **Active Learning**: Pause to think and solve problems
- **Community**: Join discussion forums (Reddit, Discord)
- **Practice**: Code along with examples

### Recommended Order for Self-Learners

**Beginner Path**:
1. CS106A (Programming)
2. CS106B (Data Structures)
3. CS107 (Systems)
4. CS221 (AI Intro)

**ML Path**:
1. CS229 (Machine Learning)
2. CS224N (NLP) or CS231N (Vision)
3. CS234 (Reinforcement Learning)
4. CS230 (Deep Learning)

**Systems Path**:
1. CS107 (Computer Organization)
2. CS144 (Networking)
3. CS143 (Compilers)
