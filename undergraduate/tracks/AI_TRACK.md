# Artificial Intelligence Track - Stanford CS

**Undergraduate Specialization in AI/ML**

---

## 🎯 Track Overview

The AI track prepares students for careers in machine learning, deep learning, natural language processing, computer vision, and robotics. This is one of the most popular tracks at Stanford.

### Prerequisites
- All core CS courses (CS106A/B, CS107, CS110, CS103, CS109, CS161)
- Linear algebra (MATH 51 or CS 205A)
- Multivariable calculus (MATH 51 or CME 100)

---

## 📚 Required Courses (Choose 5-7)

### Foundation Courses

#### **CS221 - Artificial Intelligence: Principles and Techniques**
- **Units**: 3-4
- **Prerequisites**: CS103, CS106B, CS109
- **Quarter**: Autumn, Spring
- **Instructor**: Percy Liang, Dorsa Sadigh
- **Topics**:
  - Search algorithms (A*, minimax, alpha-beta pruning)
  - Markov decision processes (MDPs)
  - Reinforcement learning basics
  - Bayesian networks
  - Machine learning fundamentals
  - Logic and reasoning
- **Workload**: 15-20 hours/week
- **Projects**: 6 programming assignments + final project
- **Languages**: Python
- **Key Concepts**:
  - State-space search
  - Probabilistic reasoning
  - Learning from data
  - Decision making under uncertainty
- **Learning Outcomes**:
  - Implement AI algorithms from scratch
  - Model real-world problems as search/optimization
  - Apply probabilistic reasoning
  - Build intelligent agents

#### **CS229 - Machine Learning**
- **Units**: 3-4
- **Prerequisites**: CS106B, MATH 51, CS109 (or STATS 116)
- **Quarter**: Autumn, Winter, Spring, Summer
- **Instructor**: Andrew Ng, Chris Ré
- **Topics**:
  - Supervised learning (regression, classification)
  - Unsupervised learning (clustering, PCA)
  - Deep learning fundamentals
  - Neural networks and backpropagation
  - Support vector machines (SVMs)
  - Decision trees and ensemble methods
  - Bias-variance tradeoff
  - Regularization techniques
  - Optimization algorithms
- **Workload**: 15-20 hours/week
- **Projects**: 5 problem sets + final project
- **Languages**: Python (NumPy, scikit-learn, TensorFlow)
- **Textbook**: *The Elements of Statistical Learning* (Hastie, Tibshirani, Friedman)
- **Learning Outcomes**:
  - Implement ML algorithms from scratch
  - Apply ML to real-world datasets
  - Evaluate and tune models
  - Understand theoretical foundations

## 💼 Career Paths

### Industry Roles
- **Machine Learning Engineer**: Build and deploy ML models
- **Research Scientist**: Conduct AI research in industry
- **Data Scientist**: Analyze data and build predictive models
- **Computer Vision Engineer**: Build vision systems
- **NLP Engineer**: Build language understanding systems
- **Robotics Engineer**: Build autonomous systems

### Companies
- Tech Giants: Google, Meta, Microsoft, Apple, Amazon
- AI Startups: OpenAI, Anthropic, Cohere, Hugging Face
- Autonomous Vehicles: Tesla, Waymo, Cruise
- Robotics: Boston Dynamics, Figure AI
- Healthcare AI: Tempus, Freenome

### Graduate School
- **MS in CS (AI Track)**: 1-2 years, coursework-focused
- **PhD in CS (AI)**: 4-6 years, research-focused
- Top programs: Stanford, MIT, CMU, Berkeley, UIUC

## 💡 Tips for Success

### Course Selection
- Take CS221 and CS229 early (foundation courses)
- Choose specialization (NLP, Vision, RL, etc.)
- Balance theory and applications
- Take at least one project-based course

### Project Ideas
- Implement papers from top conferences
- Participate in Kaggle competitions
- Build AI applications (chatbots, image generators, etc.)
- Contribute to open-source AI projects

### Networking
- Attend AI talks and seminars
- Join Stanford AI Lab events
- Participate in hackathons
- Connect with professors and researchers

### Staying Current
- Follow AI researchers on Twitter/X
- Read papers on arXiv
- Watch conference talks on YouTube
- Subscribe to AI newsletters (The Batch, Import AI)
