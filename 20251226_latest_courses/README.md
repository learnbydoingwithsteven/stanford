# Stanford AI Course Materials (2025 Collection)

This directory contains slides and notes for the latest available Stanford AI courses.
The following courses are included:

## 1. CS224N: Natural Language Processing with Deep Learning
- **Term**: Winter 2025 (Slides), Winter 2023 (Notes)
- **Content**: Transformers, RNNs, LLMs, RLHF, DPO.
- **Source**: [web.stanford.edu/class/cs224n/](https://web.stanford.edu/class/cs224n/)

## 2. CS231n: Deep Learning for Computer Vision
- **Term**: Spring 2024
- **Content**: CNNs, Vision Transformers, Generative Models, Video Understanding.
- **Source**: [cs231n.stanford.edu](http://cs231n.stanford.edu/)

## 3. CS229: Machine Learning
- **Term**: Fall 2021 (Latest accessible public materials)
- **Content**: Supervised/Unsupervised Learning, RL, Deep Learning foundations.
- **Source**: [cs229.stanford.edu](https://cs229.stanford.edu/)

## 4. CS224W: Machine Learning with Graphs
- **Term**: Fall 2024
- **Content**: GNNs, Graph Transformers, Knowledge Graphs, Reasoning.
- **Source**: [web.stanford.edu/class/cs224w/](http://web.stanford.edu/class/cs224w/)

## 5. CS324: Large Language Models
- **Term**: Winter 2022
- **Content**: Scaling Laws, Capabilities, Harms, Training, Adaptation.
- **Source**: [stanford-cs324.github.io](https://stanford-cs324.github.io/)

## 6. CS25: Transformers United
- **Term**: Spring 2024
- **Content**: Guest lectures on latest Transformer applications.
- **Source**: [web.stanford.edu/class/cs25/](https://web.stanford.edu/class/cs25/)

## Usage
The `download_materials.py` script in this directory scrapes and downloads these materials.
- HTML Notes are converted to Markdown.
- Slides are saved as PDFs.
